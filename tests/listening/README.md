# 托福听力组件测试

## 测试组件
1. `ListeningPlayer.js` - 音频播放器组件
2. `ListeningQuestion.js` - 听力题目组件

## 测试方法

### 方法一：使用Vite开发服务器（推荐）
```bash
# 在项目根目录运行
npm run dev
# 或
npm start
```

然后在浏览器中访问：
```
http://localhost:5173/tests/listening/test-listening.html
```

### 方法二：直接打开HTML文件（有限制）
由于ES模块导入和CORS限制，直接打开HTML文件可能无法正常工作。如果必须直接打开，请确保：
1. 在Chrome中启用本地文件访问：`chrome://flags/#allow-insecure-localhost`
2. 或使用本地HTTP服务器：`npx serve .`

## 组件API验证

### ListeningPlayer API
- `loadAudio(url)` - 加载音频
- `play()` - 播放音频
- `pause()` - 暂停音频
- `seek(time)` - 跳转到指定时间
- `render(container, DOM)` - 渲染UI
- `destroy()` - 销毁组件

### ListeningQuestion API
- `render()` - 渲染题目
- `setAudioPlayed(isPlayed)` - 设置音频播放状态
- `getUserAnswer()` - 获取用户答案
- `showAnswer()` / `hideAnswer()` - 显示/隐藏答案
- `reset()` - 重置题目状态
- `destroy()` - 销毁组件

### 故障排除

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 组件无法加载 | 路径错误 | 检查相对路径是否正确 |
| 音频无法播放 | CORS限制 | 使用在线音频或本地服务器 |
| 样式不正常 | CSS未加载 | 检查DOM工具是否正确导出 |
| 交互无响应 | 事件未绑定 | 检查组件初始化是否正确 |
