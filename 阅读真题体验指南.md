# TOEFL iBT 阅读模拟软件 - 体验指南

## 项目概述

这是一个功能完整的TOEFL iBT阅读模拟软件，包含多种题型和现代化的交互功能。系统采用模块化设计，具有计时器、答案检查、复习面板等辅助功能。

## 核心功能特性

### 1. 计时系统

- 跨页面同步的11分30秒倒计时器
- 可隐藏/显示计时器
- 状态通过localStorage自动保存和恢复

### 2. 导航系统

- 统一的顶部导航栏
- Volume、Help、Review、Check Answers、Back、Next等按钮
- 流畅的题目间导航

### 3. 答题功能

- **填空题**（Question 1-10, 11-20）：
  - 使用"豆包方案"的字母方框输入系统
  - 支持智能导航、自动跳转、粘贴和答案检查
  - 每个字母独立输入框，支持键盘导航
- **选择题**（Question 21, 22）：
  - 针对Email阅读题型的单选功能
  - 具有苹果风格的UI设计
  - 清晰的选项布局

### 4. 辅助功能

- **Check Answers**：检查答案正确性，显示统计信息（总数、正确、错误、准确率）和每题详情
- **Review面板**：侧边栏形式的题目概览面板，显示答题进度、标记状态，支持跳转到特定题目
- **自动保存**：用户输入的答案通过localStorage自动保存和恢复

## 完整体验流程

### 步骤1：启动系统

1. 打开 `reading_start_page.html`（阅读部分起始页）
2. 阅读三种任务类型的说明
3. 点击 **Begin** 按钮

### 步骤2：模块介绍与计时器启动

1. 进入 `module1-intro.html`
2. 阅读模块说明
3. 点击 **Begin** 启动11分30秒计时器

### 步骤3：填空题组1（Questions 1-10）

1. 进入 `reading_question1.html`
2. **题目内容**：关于古生物学（Paleontology）的短文填空
3. **操作**：
   - 在字母方框中输入答案
   - 使用Tab键或方向键在输入框间导航
   - 可点击**Review**按钮查看题目概览
   - 可点击**Check Answers**检查答案
4. 完成所有填空后，点击 **Next** 进入下一题组

### 步骤4：填空题组2（Questions 11-20）

1. 进入 `reading_question2.html`
2. **题目内容**：关于真菌（Fungi）的短文填空
3. **操作**：同上
4. 完成所有填空后，点击 **Next** 进入选择题

### 步骤5：选择题1（Question 21）

1. 进入 `reading_question3_1.html`
2. **题目内容**：Email阅读题型 - 关于会议安排的邮件
3. **操作**：
   - 阅读邮件内容
   - 选择最符合问题描述的选项
   - 可点击**Review**或**Check Answers**
4. 点击 **Next** 进入下一选择题

### 步骤6：选择题2（Question 22）

1. 进入 `reading_question3_2.html`
2. **题目内容**：Email阅读题型 - 关于项目更新的邮件
3. **操作**：同上
4. 点击 **Finish** 返回起始页面，完成体验

## 模块化设计亮点

### 组件化架构

- `components/` 目录包含可重用的UI组件：
  - `header.html` - 统一导航栏
  - `review-panel.html` - 复习面板
  - `answers-check-area.html` - 答案检查区域
  - `question-container.html` - 题目容器
  - `email-container.html` - 邮件容器
  - `progress.html` - 进度指示器

### 状态管理

- 使用localStorage在页面间同步：
  - 计时器状态 (`toefl_timer_remaining`, `toefl_timer_started`)
  - 用户答案 (`toefl_answers_module1`)
  - 标记题目 (`toefl_marked_questions`)

### 现代化前端架构

- 同时提供传统多页HTML应用和现代化单页应用
- 使用Vite构建工具
- 支持Electron桌面应用
- 模块化的JavaScript代码结构

## 测试建议

### 功能测试点

1. **计时器功能**：验证计时器是否在页面间同步，是否可隐藏/显示
2. **答案输入**：测试填空题的字母输入、导航、粘贴功能
3. **答案检查**：验证Check Answers功能的统计信息和详情显示
4. **复习面板**：测试题目跳转和标记功能
5. **答案保存**：验证刷新页面后答案是否恢复
6. **导航流程**：测试所有页面间的正向和反向导航

### 用户体验测试

1. 从起始页到完成页的完整流程
2. 各种屏幕尺寸的响应式布局
3. 键盘导航和辅助功能
4. 错误处理和用户提示

## 技术栈

- **前端**：HTML5, CSS3, JavaScript (ES6+)
- **构建工具**：Vite
- **桌面支持**：Electron
- **代码质量**：ESLint, Prettier
- **图标库**：Font Awesome 6.4.0

## 快速开始

1. 双击 `reading_start_page.html` 在浏览器中打开
2. 或运行 `start-server.bat` 启动本地HTTP服务器
3. 按照上述体验流程测试所有功能

## 注意事项

- 所有用户数据保存在浏览器localStorage中
- 刷新页面不会丢失进度（计时器、答案等）
- 支持现代浏览器（Chrome, Firefox, Edge等）
- 如需重置数据，可清除浏览器localStorage
