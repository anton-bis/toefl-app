## 技术要点

- 打字练习需要实时监听 `input` 事件比对原文，类似 Fill Blank 的输入系统
- 单词打卡需要词库 JSON + 随机抽取算法 + 答题记录存储
- 两个功能都需要在生成器 `generate_toefl_pages.js` 中新增模板和生成逻辑
- 主页侧边栏需要在生成器的 index.html 模板中增加 "Skills" 分区
