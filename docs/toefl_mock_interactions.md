## P0-1：题目页面的完整呈现与交互

### 交互流程

```
用户进入题目页
    │
    ▼
页面加载 → 解析DOM → 聚焦第一个输入框
    │
    ▼
用户点击输入框 → 输入字母 → 自动过滤非英文字符
    │
    ├── 输入满格 → 自动跳转下一格
    │
    ├── 按Tab → 跳转下一格
    │
    ├── 按Backspace(空) → 删除前格内容并聚焦
    │
    ├── 按方向键 ← → → 跳转相邻格
    │
    └── 粘贴英文字符 → 自动填入当前及后续空格
    │
    ▼
计时器倒计时中 / 用户持续作答
    │
    ▼
点击Next → 保存当前答案 → 跳转下一页
```

## P0-2：计时器跨页连续与计时准确性

### 交互流程

```
页面A加载
    │
    ▼
检查 localStorage['toefl_timer_started']
    │
    ├── 未设置（首次进入）→ 使用模板注入的TIMER_SECONDS（690/540）
    │
    └── 已设置 → 读取 localStorage['toefl_timer_remaining']
              │
              ▼
          恢复计时器秒数
              │
              ▼
计时器Interval启动 → 每秒-1 → 更新显示
    │
    ▼
页面切换（Next/Back/跳转）
    │
    ▼
离开前：localStorage.setItem('toefl_timer_remaining', totalSeconds)
localStorage.setItem('toefl_timer_started', 'true')
    │
    ▼
新页面加载 → 恢复计时器 → 继续倒计时
```

## P0-3：题目间的Next/Back导航链接正确跳转

### 交互流程

```
用户点击 Next 按钮
    │
    ▼
1. 保存当前页答案到 localStorage
   localStorage.setItem('toefl_answers_module1', JSON.stringify(answers))
    │
2. 保存计时器状态
   localStorage.setItem('toefl_timer_remaining', totalSeconds)
   localStorage.setItem('toefl_timer_started', 'true')
    │
3. 短暂禁用按钮（防重复点击）
   nextBtn.disabled = true
    │
4. 跳转目标页面
   window.location.href = '{{NEXT_PAGE_URL}}'
```

## P0-4：结果页正确统计并展示模块维度信息

### 交互流程

```
用户进入 Results 页
    │
    ▼
页面加载 → 解析 localStorage
    │
    ├── 读取 toefl_answers → 计算已答数量
    ├── 读取 toefl_marked_questions → 获取标记列表
    └── 清除计时器数据（测试已完成）
    │
    ▼
计算统计数据
    │
    ├── 总题数 = MODULE1_TOTAL + MODULE2_TOTAL
    ├── 已答数 = Object.keys(answers).length
    ├── 正确数 = 对比 answers 与 正确答案（如有）
    ├── 准确率 = 正确数 / 总题数 * 100%
    └── 用时 = 从计时器记录推算（可选）
    │
    ▼
渲染结果卡片 + 模块题目网格
```


## P1-6：Check Answers功能的核对与反馈

### 交互流程

```
用户点击 Check Answers 按钮
    │
    ▼
遍历所有 word-fill-container
    │
    ▼
获取每题的正确答案（data-answer 属性）
    │
    ▼
比对用户输入与正确答案
    │
    ├── 完全匹配 → 标记为 correct
    ├── 有输入但不匹配 → 标记为 incorrect
    └── 无输入 → 标记为 empty
    │
    ▼
更新统计区域
    │
    ├── Correct: N
    ├── Incorrect: N
    └── Accuracy: N%
    │
    ▼
生成答案详情列表
    │
    ▼
显示答案检查区域 + 平滑滚动到该区域
```

## P1-8：自动保存与刷新恢复

### 交互流程

```
页面加载
    │
    ▼
initializeAutoSave()
    │
    ├── 加载已保存答案 → loadSavedAnswers()
    └── 启动定时保存 → startAutoSaveTimer()
            │
            ▼
        每3秒执行 saveCurrentAnswers()
                │
                ▼
            遍历所有 letter-box
                │
                ▼
            收集用户输入 → answers 对象
                │
                ▼
            保存到 localStorage
            localStorage.setItem('toefl_answers_module1', JSON.stringify(answers))
                │
                ▼
            显示保存通知（3秒后自动消失）
```
